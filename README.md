# Homebrew Tap for FunctionFly CLI

This repository contains the Homebrew tap for the FunctionFly CLI (`ff`).

## Installation

To install the `ff` CLI using Homebrew, run the following commands:

```bash
brew tap functionfly/tap
brew install ff
```

## Updating

To update the `ff` CLI to the latest version, run:

```bash
brew upgrade ff
```

## About

FunctionFly CLI allows you to go from idea to global API in under 60 seconds. 
For more information, visit [functionfly.com](https://functionfly.com) or check out the main repository at [github.com/functionfly/ff-cli](https://github.com/functionfly/ff-cli).
